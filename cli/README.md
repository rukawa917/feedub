# feedub CLI

Self-hosted Telegram feed reader with AI insights.

## Quick Start

```bash
pip install feedub
feedub init    # Setup wizard
feedub run     # Start everything
```

## Commands

| Command | Description |
|---------|-------------|
| `feedub init` | Interactive setup wizard |
| `feedub run` | Start all services |
| `feedub stop` | Stop all services |
| `feedub status` | Show service health |
| `feedub logs [service]` | Tail logs |
| `feedub upgrade` | Upgrade to latest version |
| `feedub backup` | Export database |
| `feedub restore <file>` | Restore from backup |
| `feedub uninstall` | Complete removal |
| `feedub config` | Show or edit config |
| `feedub version` | Show version |
